# FlashCart Partner Portal - Business Management Dashboard

> **Partner Portal** | Manage your store, orders, menu, and analytics on Bangladesh's free delivery platform.

## About

The FlashCart Partner Portal is the business management dashboard for FlashCart delivery partners. It provides real-time order notifications with a 9-layer notification system, menu management, analytics, SEO optimization tools, and certificate earning programs.

## Features

- Real-time order alerts (9 notification layers including OneSignal push)
- Menu and product management with ImgBB image uploads
- Order Kanban board with live status updates
- Analytics dashboard with SVG charts
- SEO optimization guide with live Google preview
- Certificate earning programs
- Multi-outlet support
- Invoice PDF generation
- Review management with reply system

## Tech Stack

Same as FlashCart Main, plus:
- **Push Notifications:** OneSignal Web Push SDK
- **Audio Alerts:** Web Audio API
- **Real-time:** Firebase RTDB with onValue listeners
